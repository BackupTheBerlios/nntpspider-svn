NNTPspider - Automated search for public accessable nntp servers

  • Inhalt von servermessage auf : überprüfen (durch ; ersetzen)
  • commandline options
  • html-filter für Ergebnissliste
  • die aktuellsten Postings auslesen (die mit den höchsten nummern) (Aber: dann werden es bei allen Servern die gleichen werden.)
  • GUI schreiben.
  • Dateien nicht kommentarlos überschreiben, sondern durchnummerierte Kopien anlegen (wie wget)
  • Debian-Paket
  • ordentliches Makefile

Availability

The latest version of this program can be found at the
   download area of http://developer.berlios.de/projects/nntpspider

can be found in the man page, the README and the TODO.
